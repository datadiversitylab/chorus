# Chorus
Chorus was designed to support those events when you want the room to actually participate and not just watch. Instead of a show of hands or a static survey, chorus lets a group vote, suggest answers, and see results in real time. The organizer controls the pace: post a question, add their own answers if needed, lock suggestions when needed, or push everyone to the next topic with a single click. Guests join with a four-character token, suggest answers to particular questions and see results. Nothing is stored permanently.
